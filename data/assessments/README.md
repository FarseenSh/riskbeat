# Analyst assessments

Prose risk-assessment notes curated by the maintainer (and, via PRs, the community) — **observations
with sources, never verdicts** (see [`CHARTER.md`](../../CHARTER.md): RiskBeat
produces no score or judgment of its own; that applies to humans too).

Rules of the genre:

- Every factual claim carries a source link and an as-of date.
- Feeds are quoted **verbatim** and attributed — never paraphrased into
  stronger or weaker language.
- "Not covered by any feed" is a statement about the feeds, not the protocol.
- Unconfirmed things are said to be unconfirmed. No guessing, no advice,
  no predictions.

A file containing the `<!-- TEMPLATE` marker is an unfilled template and does
not render on the site. Once real content replaces the marker, the note
appears on the protocol's detail page tagged `[curated]`, with authorship
visible in the file's git history.
